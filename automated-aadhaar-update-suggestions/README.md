## Automated Aadhaar Update Suggestions

This project uses anonymised Aadhaar data to automatically suggest required updates
like address change or mobile linking using data-driven rules.

### Tech Stack
- Backend: Python, Flask
- Frontend: React
- Containerization: Docker

### How to Run
1. Backend:
   cd backend
   python app.py

2. Frontend:
   cd frontend
   npm install
   npm start
